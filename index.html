Salvage Search v1.0

Salvage Search is a browser plugin that makes it easy to search inventory websites for vehicles. Select your browser below to install it. A search icon will appear in your toolbar.

To use the plugin, copy a VIN (ctrl-C or right click>copy), then click the toolbar button. Search result tabs will open all at once.

To change the settings on Firefox, right click the icon, select "manage extension," then click the options tab. On Chrome, right click the icon and select "options."
