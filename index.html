Salvage Search v2

Salvage Search is a browser plugin that makes it easy to search inventory websites for vehicles and download high-def images. Select your browser below to install it.

HOW TO USE

This plugin adds an extension to your toolbar, and should be fairly self-explanatory:

Type a VIN in the text box to enable the SEARCH button. Notifications will appear as Copart, IAAI, and Row52 are searched. If a match is found, it will open in a new tab. If a vehicle cannot be found on those sites, the extension will also search archive sites like Poctra.com.

The second button, the DOWNLOAD IMAGES button, only enables on Copart, IAAI, and Poctra listing pages. Even if a vehicle is no longer listed on Copart or IAAI's website, it's often possible to download images if you happen to know the stock number. If you click "download images" while a Poctra listing is open, the extension will try and access images from the original salvage yard. That's right! No more third-party watermarks.

Copart loads very quickly, but IAAI will take a little longer, and a progress bar will be displayed. If you click away from the popup while downloading IAAI images, the download may be canceled.

You can also click the GEAR BUTTON to edit your settings. Right now, you can turn salvage yards on or off.


This extension has become something I'm proud of! I've worked out a lot of bugs, but everyone uses their computer differently, and something might break. Salvage yard websites aren't designed to make this kind of thing easy, and they might change something without warning. If something doesn't work, or doesn't work the way you'd like, get in touch. REQUIRED PERMISSIONS

When you install the extension, it will ask for permissions. Here's what it's doing with them:

  • Access data on copart.com: Allows the extension to see which Copart listing you have open to know what images to fetch, and to open image tabs directly behind it.
  • Access data on iaai.com: Allows the extension to see which IAAI listing you have open, to grab image IDs from the listing, and then to download images after processing.
  • Access data on avis.aai.com: Allows the extension to get high resolution images, which are accessed from a different domain.
  • Access data in two other domains: Allows the extension to search Row52.com and Poctra.com. Also allows us to search Copart and IAAI based on a Poctra listing.
  • Get data from the clipboard: Allows the extension to pre-fill the VIN search field if you've already copied a VIN.
  • Download files and read and modify the browser's download history: Allows the extension to send images to your downloads folder. We re-name images to eliminate a potential source of PII, and we don't require you to go through a dozen save-as dialogues. No downloads history is accessed, and no monitoring is performed.
  • Access browser tabs: Allows the extension to pull lot/stock/image ID numbers from the currently active tab. In the future, there might be an option to download images from all open tabs. No data is accessed until you click one of the buttons in the pop-up.
